## Array

```java
// 声明数组
double[] arr;

// 创建数组
arr = new double[N];

// 初始化数组
for (int i = 0; i < N; i++) {
    arr[i] = 0.0;
}
```

简化写法：`double[] arr = new double[N];`

声明初始化：`int[] arr = { 1, 1, 2, 3, 5, 8 };` 这里就是声明、创建并初始化一个数组。

初始化方法：

```java
int m = 5, n = 10;

// 初始化一个大小为 10 的 int 数组
int[] nums = new int[n];

// 初始化一个 m * n 的二维布尔数组
// 其中的元素默认初始化为 false
boolean[][] visited = new boolean[m][n];
```

一般题目会以函数参数的形式传入，要在函数开头做一个非空检查，然后用索引下标访问其中的元素即可：

```java
if (nums.length == 0) {
    return;
}

for (int i = 0; i < nums.length; i++>) {
    // 访问 nums[i]
}
```
